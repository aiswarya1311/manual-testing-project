# Manual Testing Project – Nellikuzhi Furniture Website

## Project Overview
This project involves manual testing of a live e-commerce furniture website to identify functional and UI-related defects.

## Scope of Testing
The following modules were tested:

- Home Page
- Navigation Bar
- Account
- Product & Category
- Cart
- Checkout

## Testing Performed
- Functional Testing
- UI Testing
- Positive Testing
- Negative Testing

## Project Deliverables
- Test Cases
- Bug Report
- Test Summary Report
- Defect Distribution Charts
- Screenshots/Evidence

## Tools Used
- Microsoft Excel
- Google Chrome
- Manual Testing Techniques

## Project Summary
- Total Test Cases: 170
- Total Defects Reported: 21
- Real-time testing performed on a live website
